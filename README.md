# High-Level Computer Vision

This repository contains the code & assignments for the High-Level Computer Vision course. Each folder represents a different module or assignment.

## Structure

The repository is organized by assignment/project topic:

- `Assignment_1/`: Content-based image retrieval, histogram generation, and distance metrics.
- `Assignment_2/`: Deep Learning fundamentals, including manual implementation of neural networks and use of deep learning libraries.
- `Assignment_3/`: Deep CNN architectures, normalization techniques, data augmentation strategies, dropout regularization schemes and VGG-16 transfer learning.
